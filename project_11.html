Trading Application Based On Tweets

Skills Improved: Software Design, Python, Java, Android Studio, App-Design

A simple web application which invests in stocks based on tweets analyzed from the company's top executives. It uses a sentimental analysis API to rank the tweet on a scale of 1-10: proceeding to invest if the sentiment is high and vice versa. For the purpose of demonstaration, Elon Musk's tweets were used to demonstrate the app. His recent tweets were fed to a sentimental analyzer which in turn decided to invest in the stock. Within few hours, the stock rose and a profit was observed. This novel idea won us third place at one of the largest hackathons (Hack the 6ix) in Toronto.
