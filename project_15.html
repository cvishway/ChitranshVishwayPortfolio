Part Sorter (Prototype Of Assembly Line)


Skills Improved:Software Design, C, Arduino, Sensor-calibration

Using an EVShield, Arduino, and a LEGO Kit, an autonomous system was built to sort two blocks of size 1’’x1.5’’x0.5’’ into cups from a conveyor belt. Using the LEGO light sensor along with the SumoEyes sensor, a 4’’x5’’x10’’ tower-like structure was made using standard LEGO parts to process the blocks into their respective beer cups. The Sumoeyes sensor detected an incoming block 2 ft before the processing tower. Upon relaying the incoming block to the tower, the block entered the tower using a guided-ramp system to fall above the light sensor at which the color, red or blue, was established to determine the direction of the swiper attached to a motor. Sensory feedback was used to determine the direction in which the motor sorted the blocks. The swiper spun 40° CCW to knock a red block in its corresponding beer cup and vice versa. If a red block was determined, then the swiper turned towards the left to knock the block into its designated cup, and vice versa.

This project simulated a very basic manufacturing environment in which a mechatronics system was utilized to organize a set of parts. By utlizing sensors and building blocks, I was able to learn how to create the software, design the mechanical body, and calibrate the mechatronics system to complete the task successfully.
