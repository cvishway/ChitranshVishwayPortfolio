Car Software for Crossing Maze With 0 Collisions

Skills Improved: C, Software Design, Object Oriented Programming (OOP), Sensors, Control Systems


A MechBot was coded to complete a Maze with minimal collisions and fast times. Three infrared sensors, along with 4 line following sensors on the bot were used to sense the MechBotâ€™s surroundings and exit the maze under 25 seconds with minimal collisions. Arduino Uno was used as the microcontroller with an Atmega328 microchip. Altogether, the microcontroller used 7 sensors : 4 line following and 3 infrared to sense its environment and adjust itself. Closed Loop feedback system was utilized to coontrol the car. Open loop systems were also embedded within the path to minimize time, however, they were triggered by initial sensor readings to eliminate false-trigger errors. The closed loop system shown is the skeleton model of completing the tasks. C programming language was used to code the arduino bot. Object Oriented Programming was implemented to reduce run-time of the software.
