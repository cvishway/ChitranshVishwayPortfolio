PLC Manufacturing Process for Small Flashlights

Skills Improved: PLC, Manufacturing, Robotics, Sensors



A PLC system was designed with 6+ workstations to manufacture a push button light with the aid of sensors and robots. The process was thoroughly developed using a flow-chart to eliminate any logic errors. After, the flowchart was translated into PLC code to control each indivdial workstation. A GUI (Graphical User Inferface) was developed to show a visual representation of the process and parts.
